# Roadmap

This file tracks open strategic work items for the project.

## Open strategic items

- [ ] Choose final independent product name
- [ ] Prepare v0.1.0 release after licensing cleanup
- [ ] Design public/private architecture split for future AI agent product
- [ ] Define pet bundle format and asset licensing rules
- [ ] Investigate macOS app signing and notarization
- [ ] Evaluate plugin system for community pets

## Scope notes

This repository is the open-source macOS desktop pet runtime. The roadmap
items above relate to strategic decisions about the project's future. Not all
items will result in changes to this public repository.

Future AI-agent behavior, cloud features, premium characters, packaged app
distribution, and commercial product work will be developed separately from
this open-source runtime.
